AROUND THE WEB

Projects

Teacher's Pet

Teacher's Pet is an open-source web application that allows teachers to do track student engagement during Zoom calls. Students run the app alongside their Zoom class, collecting information about whether theyre looking at the screen and their emotions. The teacher can then use this data to measure engagement and general feedback, such as how many kids showed confusion or happiness throughout class. I worked with a small team to build this project for the MLH Fellowship's Opening Hackathon. Teacher's Pet uses a variety of open-source technologies including GazeTracking, OpenCV, Keras, and Flask.

Equity

Equity is a chrome extension for Amazon.com that provides ethical scores for products. The scores come from The Ethical Consumer, an independent, not-for-profit, co-operative, based in the United Kingdom. I worked on Equity with a small group for CalHacks 6.0. Built with Python, Javascript, HTML, CSS, Beautiful-soup, Json

NumPy Dupe

I implemented a subset of the NumPy module using optimization techniques and the Python-C interface. My version was 60 times faster than the naive implementation for matrix operations. Built with C, Python, OpenMP, Intel AVX Intrinsics

Movie Genre Classifier

I used k-nearest neighbors to build a classifier that predicts a movie's genre using the frequency of words in its screenplay with 80% accuracy. Built with Python and Jupyter Notebook.

61CPU

I built a pipelined CPU from scratch in Logism. This included wiring up an arithmetic logic unit, a register file, a control logic unit, an immediate generator, and branch comparator. I then wrote and ran RISC-V code (an open standard instruction set architecture) on my CPU, fun stuff.
